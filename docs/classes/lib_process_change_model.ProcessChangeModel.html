Private changeTrackingEdges

changeTrackingEdges: { new: null | BPMNNode; old: null | BPMNNode }[] = []
  • Defined in lib/process-change-model.ts:53

Basically the 'delta' between two processes. An array of edges pointing from the old bpmn node to the new bpmn node, if existent. If the old or new node does not exist (elements were added/removed), it is null in the changeTrackingEdge.
